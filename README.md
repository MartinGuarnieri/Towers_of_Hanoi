# Towers_of_Hanoi
PyQt5 based Towers of Hanoi Game \n
Run Hanoi_Game.py on the command line to play the GUI version \n
    $ python3 Hanoi_Game.py \n
This requires PyQt Python Library: \n
    $ pip install PyQt5 \n
Run python3 Hanoi_Text.py on the command line to play the simplified command line text version \n
    $ python3 Hanoi_Text.py \n
